Definition and Usage

  • The h1-h6 tags are used to define HTML Headings.
  • h1 defines the most important heading and h6 defines the least important heading.
  • Headings are block element.
  • h1 element should be used only once for a html document.
  • Other heading tags can be used more than one in a html document.
